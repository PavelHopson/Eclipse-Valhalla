## 1. Установка и первый запуск

### Windows
1. Скачайте `EclipseValhalla-Setup-X.X.X-win-x64.exe` с [GitHub Releases](https://github.com/PavelHopson/Eclipse-Valhalla/releases)
2. Запустите установщик → выберите папку → установите
3. Если Windows SmartScreen блокирует: нажмите "Подробнее" → "Всё равно выполнить"
4. Ярлык появится на рабочем столе и в меню Пуск

### Портативная версия
- Скачайте `EclipseValhalla-Portable-X.X.X-win-x64.exe`
- Запустите без установки — всё работает из одного файла

### Веб-версия
- Откройте https://pavelhopson.github.io/Eclipse-Valhalla/ в браузере
- Работает без установки, но без некоторых десктопных функций

### Первый запуск
- При первом входе создаётся гостевой аккаунт "Warrior"
- Язык определяется автоматически (EN/RU)
- Появятся подсказки для каждого раздела — нажмите "Понятно" после прочтения

## 2. Подключение AI

AI нужен для: Оракула (чат), Кузни (изображения), Голоса (TTS), обогащения новостей.

### Вариант A: Ollama (локально, бесплатно, приватно) ⭐ Рекомендуется

**Что это:** AI работает на вашем компьютере. Бесплатно, без лимитов, полная приватность.

**Требования:** 8+ ГБ RAM (16 ГБ рекомендуется), 5-20 ГБ свободного места.

**Установка:**

1. Скачайте Ollama: https://ollama.com/download
2. Установите и запустите (появится иконка ламы в трее)
3. Откройте терминал (PowerShell или cmd) и выполните:

```bash
# Быстрая модель (5 ГБ, ответ за 5-10 сек)
ollama run qwen3:8b

# ИЛИ мощная модель (20 ГБ, ответ за 30-60 сек)
ollama run qwen3:32b
```

4. Дождитесь загрузки (прогресс-бар покажет %)
5. Когда увидите `>>>` — модель готова

**Настройка в Valhalla:**

1. Откройте **Кодекс (Настройки)** → разверните **Oracle Stack**
2. Нажмите **+ Добавить**
3. Выберите тип: **OpenAI / Compatible**
4. Заполните:
   - **Название:** `Qwen 3 Local`
   - **API Key:** `ollama` (любой текст)
   - **Model:** `qwen3:8b` (или `qwen3:32b`)
   - **Base URL:** `http://localhost:11434/v1`
5. Нажмите **Сохранить**
6. Нажмите ⭐ (звёздочку) чтобы сделать основным провайдером

**Доступные модели Ollama:**

| Модель | Размер | Скорость | Для чего |
|--------|--------|----------|----------|
| `qwen3:8b` | 5 ГБ | Быстрая (5-10 сек) | Повседневные вопросы |
| `qwen3:32b` | 20 ГБ | Средняя (30-60 сек) | Сложные задачи |
| `llama3.2:3b` | 2 ГБ | Очень быстрая (2-3 сек) | Простые ответы |
| `llama3.3:70b` | 40 ГБ | Медленная (2-5 мин) | Максимальное качество |
| `codellama:13b` | 7 ГБ | Быстрая | Программирование |
| `mistral:7b` | 4 ГБ | Быстрая | Универсальная |

**Установка дополнительных моделей:**
```bash
ollama pull qwen3:8b        # Скачать модель
ollama list                  # Показать установленные
ollama rm model_name         # Удалить модель
```

**Важно:** Терминал с Ollama должен быть открыт (или Ollama запущена в фоне через трей).

---

### Вариант B: OpenRouter (облако, бесплатный tier)

**Что это:** Облачный API, объединяющий 100+ моделей. Есть бесплатные модели.

**Установка:**

1. Зарегистрируйтесь: https://openrouter.ai
2. Получите API ключ: https://openrouter.ai/workspaces/default/keys
3. Нажмите **Create Key** → скопируйте ключ

**Настройка в Valhalla:**

1. **Кодекс** → **Oracle Stack** → **+ Добавить**
2. Тип: **OpenAI / Compatible**
3. Заполните:
   - **Название:** `OpenRouter Free`
   - **API Key:** ваш ключ с OpenRouter
   - **Model:** `meta-llama/llama-3.3-70b-instruct:free`
   - **Base URL:** `https://openrouter.ai/api/v1`
4. Сохранить

**Бесплатные модели OpenRouter:**

| Модель | Качество | Примечание |
|--------|----------|-----------|
| `meta-llama/llama-3.3-70b-instruct:free` | Отличное | Самая стабильная |
| `google/gemma-4-31b-it:free` | Хорошее | Быстрая |
| `qwen/qwen3-coder:free` | Отличное | Для кода |
| `nousresearch/hermes-3-llama-3.1-405b:free` | Максимальное | Может быть медленной |

**Ограничения:** Бесплатные модели могут быть временно rate-limited (ошибка 429). Подождите 1-2 минуты и попробуйте снова, или переключитесь на другую модель.

### Вариант C: Google Gemini (бесплатно, картинки + голос)

**Что это:** Единственный провайдер с поддержкой генерации изображений и TTS.

**Установка:**

1. Откройте: https://aistudio.google.com/apikey
2. Войдите с Google аккаунтом
3. Нажмите **Create API Key** → скопируйте

**Настройка в Valhalla:**

1. **Кодекс** → **Oracle Stack** → **+ Добавить**
2. Тип: **Google Gemini**
3. Заполните:
   - **Название:** `Gemini`
   - **API Key:** ваш ключ
   - **Model:** `gemini-2.5-flash-preview-05-20`
4. Сохранить

**Лимиты:** ~1500 запросов/день (Flash), ~50 изображений/день. Ключ хранится только в вашем браузере.

---

### Вариант D: OpenAI (платно, высшее качество)

1. Зарегистрируйтесь: https://platform.openai.com
2. Пополните баланс ($5 минимум): Settings → Billing
3. Создайте ключ: https://platform.openai.com/api-keys

**Настройка:** тип **OpenAI / Compatible**, модель `gpt-4o-mini`, без Base URL.

### Несколько провайдеров одновременно

Можно добавить несколько провайдеров и переключаться между ними:
- ⭐ (звёздочка) = основной провайдер для чата
- Gemini нужен отдельно для Кузни (изображения) и Голоса (TTS)
- Ollama для быстрых вопросов, OpenRouter для сложных задач

## 3. Чертог (Dashboard)

Главный экран — ваш командный центр.

### Командный контур
- **"Система ждёт приказа"** — нет активных квестов, добавьте цель
- **"Давление держится"** — есть квесты, всё по плану
- **"Долг уже открыт"** — есть просроченные квесты, действуйте

### Дисциплина (число 12-99)
Рассчитывается автоматически:
- Базовый уровень: 48
- +34 максимум за процент выполнения
- +18 максимум за стрик (серию дней)
- -36 максимум за просроченные квесты

### Виджет "Сегодня"
4 карточки: квесты на сегодня, выполнено сегодня, серия дней, уровень.

### Быстрый ввод
Введите цель в поле ввода → Enter → квест создан с дедлайном +24 часа.

### Режим фокуса
Нажмите "Режим фокуса" → выберите квест → таймер 25 минут → полноэкранный режим без отвлечений.

Если переключите вкладку — система засчитает побег и потребует объяснений.

## 4. Битвы (Квесты)

### Создание квеста
1. Нажмите **+ Новый Поход**
2. Заполните:
   - **Цель** — название задачи
   - **Детали** — описание (опционально)
   - **Дата и время** — дедлайн
   - **Приоритет** — Low / Medium / High
   - **Категория** — Работа / Личное / Здоровье / Покупки / Финансы / Учёба
   - **Повтор** — Без повтора / Ежедневно / Еженедельно / Ежемесячно
   - **Оценка времени** — сколько минут на задачу
   - **Подзадачи** — чеклист внутри квеста

### Режимы отображения
- **Свиток** — вертикальный список
- **Тактика** — канбан-доска (TODO → IN_PROGRESS → DONE)

### Фильтры
- **Активные цели** — текущие задачи
- **Незавершённые** — все не выполненные
- **Брошенные** — просроченные
- **Запечатанные** — выполненные

### Подзадачи
Прогресс-бар на карточке показывает сколько подзадач выполнено. Нажмите на квест чтобы отметить подзадачи.

## 5. Летопись (Календарь)

### Месяц
- Сетка 7 дней, русский локаль начинается с понедельника
- Цветные точки = квесты с приоритетами
- Время отображается рядом с названием
- Сегодняшний день подсвечен синим

### Лента
- 14 дней вперёд
- Полная информация по каждому квесту
- Цветные бейджи времени

### Создание через календарь
Нажмите на любой день → откроется модалка создания квеста с этой датой.

## 6. Полигон (Тренировки)

### 6.1 Режимы — ваши программы тренировок

**Создание программы:**
1. Нажмите **Создать Режим**
2. Заполните:
   - Название (например, "Тренировка А")
   - Метка дня (например, "Push Day")
   - Видео тренировки — YouTube URL или файл с ПК
   - Время отдыха — 30/45/60/90/120/180 секунд
   - Дни недели — на какие дни назначить
3. Добавьте упражнения: название, подходы, повторы, видео

**Запуск тренировки:**
- Нажмите **Начать** на карточке программы
- Появится вкладка **Бой** — ваша активная тренировка

### 6.2 Бой — активная тренировка

- **Таймер** — считает общее время тренировки
- **Калории** — `🔥 X ккал` в реальном времени
- **Прогресс** — % выполненных подходов
- **Видео** — встроенный плеер (если видео прикреплено)

**Для каждого упражнения:**
1. Раскройте карточку (нажмите на упражнение)
2. Введите вес и повторы для каждого подхода
3. Нажмите ✓ для завершения подхода
4. Автоматически запустится таймер отдыха
5. Можно менять длительность отдыха во время тренировки

**Личные рекорды:**
Если вы подняли больший вес чем раньше — появится золотой баннер "Новый рекорд!".

**Сохранение сессии:**
Если приложение закроется во время тренировки — при следующем запуске она восстановится.

### 6.3 Планы — многонедельные программы

1. Нажмите **Новый план**
2. Введите название и выберите количество недель (2/4/6/8/12)
3. Раскройте каждую неделю → назначьте тренировки на дни
4. Выберите интенсивность: Лёгкая / Средняя / Тяжёлая / Разгрузка
5. Кнопка Copy дублирует неделю
6. Стрелки ← → переключают текущую неделю

### 6.4 Видео — видеотека

1. Нажмите **+ Добавить**
2. Введите название видео
3. Вставьте YouTube URL или нажмите **Файл** для выбора с ПК
4. Выберите категорию (Всё тело / Пресс / Спина / Ноги / и т.д.)
5. Нажмите **Добавить**
6. Клик по карточке → видео в плеере

### 6.5 Кодекс — готовые программы

10 шаблонов, фильтр по категории:

**Своим весом (без оборудования):**
- Тело Берсерка — полная закалка тела
- Железный пресс — пресс и кор
- Толкай и тяни — верх тела
- День ног (дома) — ноги со стулом
- Ноль оборудования — всё тело, где угодно

**Бег:**
- Бег: Новичок — интервалы ходьба/бег
- Бег: Выносливость — длинные интервалы

**С весами:**
- Молот Тора — руки с гантелями

**Растяжка:**
- Приветствие Солнца — утренняя разминка

**HIIT:**
- Полёт Валькирии — скорость и выносливость

### 6.6 Руны — статистика
- Тренировки, общий объём, подходы, среднее время, калории
- График объёма по тренировкам

### 6.7 Саги — история
- Хронологический список всех тренировок
- Раскройте запись для деталей по каждому подходу

## 8. Оракул (AI-ассистент)

### Быстрые действия
- **Планировать кампанию** — AI спланирует ваш день на основе квестов
- **Прочитать паттерн** — проанализирует ваши привычки и слабости
- **Сломать сопротивление** — мотивационный пинок от системы

### Что спрашивать
- Планирование: "Составь план на неделю для набора массы"
- Продуктивность: "Как перестать прокрастинировать?"
- Тренировки: "Программа тренировок дома без оборудования"
- Питание: "Рацион на 2500 ккал для набора массы"
- Код/работа: "Объясни React hooks простыми словами"
- Мотивация: "Дай 5 привычек для дисциплины"

### Форматирование
Ответы отображаются с markdown: жирный текст, заголовки, списки, разделители.

### История
Разговор сохраняется между переключениями вкладок. Кнопка 🗑️ очищает историю.

## 10. Зал Славы (Достижения)

### 31 достижение в 5 категориях

**Битвы (квесты):**
- Первая кровь (1 квест) → Легенда Вальхаллы (500 квестов)

**Дисциплина (стрики):**
- Первое пламя (3 дня) → Неугасимый огонь (100 дней)

**Выносливость (фокус):**
- Первый транс (1 сессия) → Мастер дзен (100 сессий)

**Интеллект (AI):**
- Первый совет (1 беседа) → Архитектор разума (200 бесед)

**Мастерство (функции):**
- Исследователь (3 функции) → Максималист (все 6)
- Хранитель рун (5 заметок), Читатель Воронов (10 новостей)
- Воин тренировок (10 тренировок), Мастер Кузни (10 изображений)
- Полиглот (переключение языка)

### Тиры
Бронза → Серебро → Золото → Легенда. Каждый даёт XP (10-500).

### Награды
16 наград разблокируются автоматически при получении достижений:
- Титулы: Посвящённый, Воин, Ветеран, Легенда Вальхаллы
- Темы: Кровь, Золото, Пустота
- Знаки: Железная Воля, Марафонец, Архитектор, Максималист

## 14. Решение проблем

### Чёрный экран при запуске
- Переустановите приложение из последнего релиза
- Проверьте что антивирус не блокирует

### AI не отвечает
- Проверьте что Ollama запущена: откройте http://localhost:11434 в браузере
- Для OpenRouter: проверьте ключ и модель в настройках
- Ошибка 429: модель перегружена, подождите 1-2 минуты

### Данные потерялись
- Данные хранятся в `%AppData%\eclipse-valhalla\`
- Не нажимайте "Сброс" в окне ошибки — это удалит данные
- Регулярно экспортируйте данные через Настройки → Данные → Экспорт

### Приложение тормозит
- Если много квестов (100+) — попробуйте завершить старые
- Используйте модель `qwen3:8b` вместо `qwen3:32b` для быстрых ответов

### Видео не воспроизводится
- YouTube: проверьте подключение к интернету
